#  Cybersecurity Internship Project  
## Strengthening Security Measures for a Web Application

###  Overview
This project was completed during a cybersecurity internship to analyze a web-based User Management System, identify vulnerabilities, and implement basic security measures to improve application security.

---

## Week 1 – Security Assessment
- Set up a mock web application using Node.js.
- Tested signup, login, and profile modules.
- Performed vulnerability testing using:
  - OWASP ZAP
  - Browser Developer Tools
- Tested:
  - Cross-Site Scripting (XSS)
  - Basic SQL Injection
  - Security misconfigurations
- Documented identified vulnerabilities.

---

## Week 2 – Security Implementation
- Added input validation using **validator.js**
- Implemented password hashing using **bcrypt**
- Added token-based authentication using **JWT**
- Secured HTTP headers using **Helmet.js**

---

## Week 3 – Advanced Security
- Performed penetration testing using **Nmap**
- Implemented application logging using **Winston**
- Created security best practices checklist:
  - Input validation
  - Password hashing
  - Secure authentication
  - HTTPS recommendation

---

##  Tools & Technologies
Node.js | Express.js | OWASP ZAP | Nmap | Validator | Bcrypt | JWT | Helmet | Winston

---

## Outcome
- Identified and fixed common web vulnerabilities.
- Applied secure coding practices.
- Implemented logging and basic penetration testing.
- Improved overall web application security.
